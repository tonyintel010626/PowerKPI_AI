# skills

Reusable, composable skill definitions that agents can invoke.
Skills are higher-level than tools — they may chain multiple tools together.

## Examples for PowerKPI
- `generate_kpi_summary`   — pull data + summarize into narrative
- `anomaly_detection`      — run statistical check on a KPI trend
- `create_pbi_report`      — scaffold a Power BI report from a template
