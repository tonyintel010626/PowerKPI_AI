# TCSS Known Issues and Workarounds

## Overview

This document tracks known RTL bugs, driver/firmware issues, and platform-specific problems for TCSS (Type-C Subsystem) validation.

## Format

Each issue includes:
- **HSDES ID:** Sighting identifier
- **Platform:** Affected platform(s)
- **Component:** TCSS component (USB4/TBT/IOM/DP/DMA/Power)
- **Summary:** Brief description
- **Impact:** Functional impact and severity
- **Workaround:** Temporary mitigation (if available)
- **Status:** Open / Fixed / Closed
- **Fix Version:** Platform stepping or FW version with fix
