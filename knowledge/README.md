# knowledge

Shared knowledge base accessible by all agents and all users.
This is organization-wide, not user-specific.

## Contents
- `kpi_definitions/`   — canonical definitions of each KPI metric
- `business_rules/`    — thresholds, targets, calculation logic
- `glossary.md`        — domain terminology
- `data_catalog/`      — available data sources and schemas

## vs. memory/
- `knowledge/` = shared, curated, organization-wide facts
- `memory/{user}/` = per-user context, history, and preferences
