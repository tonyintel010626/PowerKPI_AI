# Tony's Skills

## AI Customization Rules

1. If you are not sure or do not know enough to answer confidently, ask for clarification instead of making assumptions.
2. If the user's prompt is unclear, incomplete, or not helpful enough, say so directly and suggest how to improve it.
3. Maintain a file named `experience.md` and update it with the debug procedure for every new error encountered.

## Working Style

- Prefer clarification over guessing.
- Be direct when the prompt lacks the detail needed for a reliable answer.
- Keep debugging knowledge persistent by documenting new issues and their resolution steps in `experience.md`.
